Goma

Rewrite is required on nginx

If you want to run Goma on this Server you have to define a correct server-configuration.

Please define the following configuration:

if (!-f $request_filename) {
        rewrite (.*) /system/application.php last;
}

# we need this for file-uploads
client_max_body_size 50M;

error_page  404  /system/application.php;
